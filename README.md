Aplicativo de Frases Aleatórias – Android

Este é um aplicativo mobile desenvolvido em Java utilizando o Android Studio, com o objetivo de exibir frases aleatórias.

✨ Funcionalidades

Exibição de uma nova frase a cada toque no botão.

Interface simples, direta e amigável.


⚙️ Tecnologias Utilizadas

Java como linguagem principal.

Android Studio como ambiente de desenvolvimento.

XML para definição de layouts.

Gradle para gerenciamento de build.

🧠 Estrutura do Projeto

O aplicativo é composto por uma única tela principal, onde o usuário pode visualizar e atualizar a frase exibida. A lógica está centralizada em uma activity principal, com as frases armazenadas localmente no próprio código ou em recursos do aplicativo.

O layout é definido com componentes básicos da interface do Android, como TextView e Button, organizados com ConstraintLayout ou LinearLayout, garantindo compatibilidade e responsividade em diversos tamanhos de tela.
